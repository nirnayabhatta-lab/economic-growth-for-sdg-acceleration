# Source Note

Core source families used in the v1 analytical layer:
- Sustainable Development Report country profiles
- World Bank World Development Indicators
- ILOSTAT
- UNDP Integrated SDG Insights
- UNDP SDG Diagnostics
- National Planning Commission Nepal documents
- ADB regional indicator compendia
- IMF macro and fiscal context reports

The source registry in `data/sources.json` is the single reference point for:
- source labels
- URLs
- quality tier
- coverage notes
- last-updated stamps
